# Screenshots

Store product screenshots in `docs/screenshots/` using these suggested names:

- `auth-login.png`
- `dashboard-overview.png`
- `projects-list.png`
- `project-details.png`
- `tasks-kanban.png`
- `analytics.png`
- `not-found.png`

Recommended export size:

- Width: 1600px
- Format: PNG
- Compression: visually lossless

Use clean, real UI states without browser chrome when possible.
